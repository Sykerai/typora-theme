# Typora 主题使用问题清单

用于记录日常使用各套 Typora 主题时发现的问题。问题先集中收集，后续有时间时再统一评估和优化，避免在记录阶段直接扩大修改范围。

## 状态说明

- `待处理`：问题已记录，尚未开始调整。
- `处理中`：正在设计或实施调整。
- `待验证`：调整已完成，等待在 Typora 中确认实际效果。
- `已完成`：调整与实际效果均已确认。
- `暂缓`：当前不处理，但保留记录。

## ISSUE-001：Phycat 暗色系列主题配色不协调

- 状态：`待处理`
- 记录日期：2026-07-16
- 影响范围：Phycat 暗色系列主题
- 问题描述：整体配色不够协调，视觉感受偏刺眼，长时间阅读时舒适度不足。
- 后续处理：集中优化时统一检查背景、正文、强调色、边框和代码语法色之间的协调性，在保留各暗色主题辨识度的前提下降低过强的饱和度与视觉刺激。
- 验证要求：调整后在 Typora 中切换多个 Phycat 暗色主题，分别检查正文、标题、表格、代码块、大纲和选中状态的实际观感。

## ISSUE-002：左侧实时大纲的高亮背景覆盖不完整

- 状态：`待处理`
- 记录日期：2026-07-16
- 影响范围：Phycat 系列主题的左侧实时大纲
- 问题描述：大纲标题较长时，节点的圆角高亮背景没有随可见文字完整延伸，部分标题文字超出背景区域，造成背景覆盖不完整。
- 后续处理：集中优化时检查大纲节点容器、标题标签与文字内容之间的宽度、最大宽度和溢出规则，使悬停及当前项背景能够完整覆盖可见标题，同时保留现有树状结构和 Phycat 配色。
- 验证要求：使用不同层级的长标题，分别检查默认、悬停和当前项状态，并确认大纲滚动时背景与文字范围始终一致。
