# Typora 主题使用问题清单

用于记录日常使用各套 Typora 主题时发现的问题。问题先集中收集，后续有时间时再统一评估和优化，避免在记录阶段直接扩大修改范围。

## 状态说明

- `待处理`：问题已记录，尚未开始调整。
- `处理中`：正在设计或实施调整。
- `待验证`：调整已完成，等待在 Typora 中确认实际效果。
- `已完成`：调整与实际效果均已确认。
- `暂缓`：当前不处理，但保留记录。

## ISSUE-001：Phycat 暗色系列主题配色不协调

- 状态：`待处理`
- 记录日期：2026-07-16
- 影响范围：Phycat 暗色系列主题
- 问题描述：整体配色不够协调，视觉感受偏刺眼，长时间阅读时舒适度不足。
- 后续处理：集中优化时统一检查背景、正文、强调色、边框和代码语法色之间的协调性，在保留各暗色主题辨识度的前提下降低过强的饱和度与视觉刺激。
- 验证要求：调整后在 Typora 中切换多个 Phycat 暗色主题，分别检查正文、标题、表格、代码块、大纲和选中状态的实际观感。
