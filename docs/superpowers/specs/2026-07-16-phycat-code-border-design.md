# Phycat 代码块外边框微调设计

## 目标

改善所有 Phycat 亮色与暗色主题中围栏代码块最外层边框的观感，使亮暗主题保持一致。

## 修改范围

- 修改共享文件 `phycat.light.css` 和 `phycat.dark.css` 中的 `.md-fences` 外层规则。
- 边框宽度由 `1px` 统一调整为 `2px`。
- 圆角统一调整为 `10px`。
- 边框颜色继续使用各主题现有变量：亮色保留 `var(--element-color-shallow)`，暗色保留现有 `color-mix(...)`。

## 不修改内容

- 不改变 Bloom Grid 两行结构、顶部栏、语言选择框、滚动方式或最大高度。
- 不改变代码字体、字号、行高、语法高亮和背景色。
- 不改变 Mermaid、表格或其他组件。

## 验证

- 检查两个共享文件的 `.md-fences` 均为 `2px` 边框和 `10px` 圆角。
- 检查花括号平衡，且现有 Grid、`70vh`、JetBrains Mono 和 16px 代码字号仍然存在。
- 写入前保留本次修改前备份。
