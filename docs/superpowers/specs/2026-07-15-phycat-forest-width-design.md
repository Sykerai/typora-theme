# Phycat Forest 正文宽度调整设计

## 目标

改善 `phycat-forest` 在高分辨率屏幕上的正文区域过窄问题，使其宽度表现与 `drake-light` 一致。

## 方案

- 将主题变量 `--max-width` 从 `950px` 调整为 `1200px`。
- 编辑视图继续通过 `max-width` 自适应：宽屏最大为 `1200px`，窄窗口随可用空间收缩。
- 在 `phycat-forest.css` 中为 HTML 导出视图添加专属覆盖规则，并引用同一 `--max-width` 变量，避免编辑与导出宽度不一致，同时不影响其他 Phycat 主题。
- 保留现有边距、内边距、侧边栏断点和打印规则，不改变主题的其他视觉效果。

## 验收标准

- 普通编辑视图的正文最大宽度为 `1200px`。
- HTML 导出页面的正文最大宽度为 `1200px`。
- 小于该宽度的窗口不产生由正文容器导致的横向滚动。
- 打印布局仍使用现有的全宽打印规则。

## 安全措施

写入前创建原主题文件的备份副本，修改后检查目标规则的最终值。共用基础文件 `phycat/phycat.light.css` 保持不变。
