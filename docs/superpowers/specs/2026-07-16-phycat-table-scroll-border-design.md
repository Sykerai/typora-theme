# Phycat 表格滚动边框一致性设计

## 问题

Phycat 表格的滚动容器和外围单元格均绘制 `.1em` 同色边框。表格位于最左端时，两层边框相邻并显得更粗；横向滚动后，表格自身的左边框随内容移出视口，只剩固定容器边框，因此粗细发生变化。

## 目标

让所有 Phycat 亮色与暗色主题中的表格外框在横向滚动前后保持相同粗细，同时保留圆角外框、内部网格和现有滚动行为。

## 修改范围

- 仅修改共享文件 `phycat.light.css` 和 `phycat.dark.css` 的表格边框规则。
- 保留滚动容器现有 `.1em` 外框，作为始终固定可见的表格边界。
- 将每行首、末单元格与外框重叠的左右边线设为 `0`。
- 将表格第一行和最后一行与外框重叠的顶部、底部边线设为 `0`。
- 亮色与暗色主题使用相同结构，继续沿用各自现有颜色变量。

## 不修改内容

- 不改变表格字号、行高、内边距、列宽计算、换行和悬停效果。
- 不改变内部横线与竖线的 `.1em` 网格结构。
- 不改变滚动条、圆角大小或 HTML 表格容器行为。
- 不修改任何单独入口主题。

## 验证

- 在暂存副本中先确认修复选择器尚不存在，再加入最小规则并确认选择器成对存在。
- 检查两个共享 CSS 的花括号平衡。
- 检查外层容器仍保留 `.1em` 边框、`overflow-x: auto` 与 `8px` 圆角。
- 检查内部单元格仍保留 `.1em` 网格边框。
- 写入实际主题前，为亮色和暗色共享文件分别创建独立备份。

## 交互验收

在 Typora 中切换主题或重启后，将超宽表格从最左端向右横向滚动，确认左侧固定外框的粗细和颜色不再发生变化，并确认表格内部网格仍连续。
